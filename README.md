# dynamoimport

jsonの内容でDynamoDBへインポートする。

# Dependency

* Python 3.7.4
* boto3 1.9.197
* tqdm 4.32.2

# Setup

* pipenv install

# Usage

1. pipenv run python dynamoimport.py テーブル名 jsonファイル
